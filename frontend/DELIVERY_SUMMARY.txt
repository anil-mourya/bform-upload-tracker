                      QUICK START GUIDE
================================================================================

INSTALLATION
  1. cd b-form-tracker
  2. npm install

CONFIGURATION
  1. cp .env.example .env
  2. Edit .env with your API URL: REACT_APP_API_URL=http://localhost:5000/api

DEVELOPMENT
  1. npm run dev
  2. Open http://localhost:3000

PRODUCTION BUILD
  1. npm run build
  2. Output in dist/ directory

DEPLOYMENT
  1. Deploy dist/ to your hosting
  2. Configure backend API URL
  3. Ensure HTTPS in production

================================================================================
                      API INTEGRATION
================================================================================

The frontend connects to backend via uploadTrackerService.js

Required Backend Endpoints:
  • GET  /api/b-forms/tracker       - Get tracker data
  • GET  /api/b-forms/stats         - Get statistics
  • POST /api/b-forms/upload        - Upload B-Form
  • POST /api/b-forms/{id}/assign   - Assign B-Form
  • GET  /api/b-forms/{id}          - Get details
  • PUT  /api/b-forms/{id}/status   - Update status
  • GET  /api/b-forms/search        - Search
  • GET  /api/b-forms/export        - Export data

Authentication: JWT Bearer Token
See API_DOCUMENTATION.md for complete specifications
